# HandballGear
HandballGear is a web project built with HTML, CSS, and JavaScript that showcases and sells handball equipment such as balls, shoes, and accessories through a clean interface.

## Project Description

**HandballGear** is an online sports store focused on **handball equipment and accessories**. The website allows users to browse various products and purchase them online if they find something they like.

Orders can be delivered to different locations across the country. The website provides information about product categories and detailed descriptions for each individual item. For some special products, additional information will be included, such as famous handball players who have used or promoted those products.

Each product will include **reviews and feedback options**, allowing users to share their experiences. The website will also include a **search functionality** that allows users to find products based on their category.

Users can add products to a **shopping cart**, which updates in real time. Additionally, visitors can learn about the motivation behind creating the **HandballGear** project.

## Similar Websites

### WePlayHandball

**Organization and Design:**  
Modern design focused on products and promotions, with large banners on the homepage and a clear category structure. Products are displayed in a grid layout.

**Pros**
- Efficient filtering by brand, size, price, and availability  
- Detailed product presentation with multiple images and technical information  

**Cons**
- Homepage may become crowded due to many promotional banners  
- Less focus on storytelling or athlete recommendations

### SportSuport

**Organization and Design:**  
Simple design focused mainly on functionality. The structure is organized by different sports categories, including handball.

**Pros**
- Intuitive navigation  
- Prices and discounts clearly displayed  

**Cons**
- Not specialized exclusively in handball  
- Simple design with limited visual elements
